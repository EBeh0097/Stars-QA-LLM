# Question and Answer NLP System for Stars Rating Year 2025 Tech-Notes Document

<h1> </h1>


<h2>Description</h2>
Generate a question answering system that answers questions about the technotes document.
Generate system by combining 4 models, 2 information retrieval models and 2 question answering models.
2 IR models are bert-base-uncased (model 1) and cross-encoder/ms-macro-MiniLm-L6-v2 (model 2), both retrieved from hugging face.
2 question answering models are deepset/Roberta-base-uncased (model 3) and distilbert-base-uncased (model 4).
